# gatsby-source-yotpo-next

> Gatsby source plugin for pulling in Yotpo reviews (with photos)

## How to install

`npm install gatsby-source-yotpo-next`

or

`yarn add gatsby-source-yotpo-next`

## How to use

```
module.exports = {
  plugins: [
    `gatsby-source-yotpo-next`
  ],
}
```
